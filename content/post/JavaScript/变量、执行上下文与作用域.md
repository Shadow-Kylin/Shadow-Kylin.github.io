# 变量
JavaScript 中的变量是松散的，它可以在初始化后再次改变类型。

JavaScript 中的变量声明使用的关键字（var、let、const）也与常见编程语言不同，其作用不是指定其类型，而是限定其作用域。

变量的类型由其值决定，变量值类型改变，变量类型随之改变。
