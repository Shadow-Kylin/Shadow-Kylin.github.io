# Service Worker 简介

Service Worker 是一种 Web 技术，是运行在用户浏览器背景中的 JavaScript 文件。它提供了拦截和处理网络请求（HTTPS 环境下）的能力，允许开发者缓存资源并创建离线体验，同时还能支持推送通知和后台数据同步等功能。
